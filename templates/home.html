Minecraft Recipes API

Base URL

http://{{ip}}/

Endpoints

  • List all Recipes
    GET /recipes/
    /recipes/
  • Search Recipes
    GET /recipes/?q={search query}
    /recipes/?q=flint and steel
  • Get random Recipe
    GET /recipes/random/
    /recipes/random/
  • Get Recipe details
    GET /recipes/{id}/
    /recipes/4415/
  • List all Recipes
    PATCH /recipes/{id}/
